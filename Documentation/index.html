CANLibrary
CAN Helper library

Introduction

This project aims to offer a simple interface through CAN network over Arduino (or other systems). Use it at your own risks.

Misc

If you need to use a different CAN BUS shield than the one I used, simply create your specific connector using the provided ICANConnector interface. You are welcome to send it to me so I can add it to the connectors list ;)

For the record, I used the MCP CAN library that you could find in the subfolders, with Arduino IDE 1.8.2, a classic Arduino UNO and the CAN BUS Shield from http://wiki.seeed.cc/CAN-BUS_Shield_V1.2/

Have fun !
